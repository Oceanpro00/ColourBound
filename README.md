# Colourbound

A beginner-friendly p5.js platformer project about restoring colour in a layered lo-fi world.

## Project Overview

Colourbound is an upcoming p5.js platformer where the player moves through simple layered levels, collects colour-based items, and gradually restores colour to the world.

The creative direction is still in development. The final character design, level themes, and exact world style are not locked in yet. The game may use a paper-like, handmade, or soft lo-fi visual style, but the current priority is building the core gameplay system first.

## Core Gameplay Idea

Each level starts with several faded or inactive colour groups.

The player collects colour items to restore those groups. When a colour group is restored, matching parts of the level become coloured again.

The player must also avoid ink-like hazards. If the player touches a hazard, one uncollected colour group is ruined and turns dark or black.

The level is complete when every colour group has either been restored or ruined.

## Core Features

- p5.js canvas-based game
- Simple player movement
- Left, right, jump, and gravity
- Platforms or layered walkable areas
- Collectible colour items
- Colour restoration system
- Hazard system that can ruin uncollected colours
- Door, exit, or level-completion trigger
- Score based on how much colour was restored

## MVP Scope

The minimum viable version will include:

- 1 playable level
- 1 simple player character
- 4 colour groups
- 4 collectible colour items
- 1 moving hazard
- Basic platform movement
- Basic collision detection
- Colour restoration logic
- Colour ruin logic
- Simple level-completion condition
- Basic score screen

## Extended Scope

If the MVP is completed successfully, the project may expand to include:

- Multiple levels
- Different visual themes per level
- More hazards
- More polished character design
- Level transitions
- Final restoration percentage
- Rank or outcome system
- Simple animation polish
